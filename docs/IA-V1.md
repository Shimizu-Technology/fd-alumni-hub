# IA — FD Alumni Tournament Hub (V1)

## Public Pages
1. Home
   - Hero (current tournament year)
   - Today's games
   - Quick actions: Watch / Buy Tickets / Standings
2. Schedule
   - Filter by date/division
   - Game cards with time/court/status
3. Standings
   - Group standings + W/L + point diff
4. Watch
   - Live now / upcoming / replays
   - Embedded or linked Clutch streams
5. News
   - Official updates + linked GSPN stories
6. Sponsors
   - Sponsor cards + links

## Admin (V1 light)
- Manage teams
- Manage schedule/results
- Toggle featured streams
- Manage sponsor placements

## UX Rules
- Mobile-first
- 2-click max to find: next game, standings, watch link
- Keep copy simple and local-context friendly
