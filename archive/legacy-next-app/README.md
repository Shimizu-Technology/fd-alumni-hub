# Legacy Next.js FD Alumni Hub App

This directory is an archive of the original Next.js/Prisma implementation that lived at `apps/web`.

It is **not deployed** and is no longer part of the active npm workspace, CI gate, or Netlify deployment path. It is retained only as historical/migration reference while the Rails API + React/Vite app becomes the active FD Alumni Basketball Hub.

Active app paths:

```text
api/  Rails API
web/  React/Vite frontend
```

If you need data migration context from this archive, prefer the documented Rails migration runbooks and scripts already copied into `docs/` and `api/lib/tasks/` before using this code directly.
